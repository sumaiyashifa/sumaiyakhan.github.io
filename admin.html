GitHub Pages Setup Guide

One-time setup to get your site live

1
Create a GitHub repo named sumaiyashifa.github.io
This exact name makes GitHub Pages serve it at your personal URL.
2
Upload these files to the repo
index.html, data.json, profile.png (your photo)
3
Enable GitHub Pages in repo Settings → Pages
Set source to main branch / root. Your site goes live at sumaiyashifa.github.io
4
Future updates: just replace data.json
Edit content in this admin panel → Export data.json → Push to GitHub. Done!
